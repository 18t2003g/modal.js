Modal.js

次のようなコードで要素をモーダルウィンドウにできます。 

                // インスタンス生成
                const modal = new Modal(document.getElementById("ID-of-element"));

                // 影をつける場合
                modal.shadow = true;

                // モーダルウィンドウを開く
                modal.show();

                // モーダルウィンドウを閉じる
                modal.hide();

をクリックするとサンプルが開きます。

その他の情報は、https://github.com/18t2003g/modal.jsを参照してください。
