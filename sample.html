Modal.js

You can make modal windows using following codes! 

                // Create Isntance
                const modal = new Modal(document.getElementById("ID-of-some-element"));

                // If you want shadow
                modal.shadow = true;

                // Open Modal Window
                modal.show();

                // Close Modal Window
                modal.hide();

to see a demo window.

More information, see https://github.com/18t2003g/modal.js.
