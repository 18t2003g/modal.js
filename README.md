# Modal.js

### About
Modal Windowを作成するためのjavacriptライブラリです。
[サンプルページ](https://18t2003g.github.io/modal.js/)

### 使用例
```
// Create Isntance
const modal = new Modal(document.getElementById("ID-of-element"));
// If you want shadow
modal.shadow = true;
// Open Modal Window
modal.show();
// Close Modal Window
modal.hide();
```
